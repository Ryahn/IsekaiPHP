# TestModule Module

A module for IsekaiPHP.

## Installation

Run `composer install` in this directory to install dependencies.

## Configuration

Edit `config/module.php` to configure the module.

## Usage

Add your module functionality here.

## Module Structure

- `Module.php` - Main module class
- `ServiceProvider.php` - Service provider for dependency injection
- `routes/` - Route definitions
- `views/` - Blade templates
- `migrations/` - Database migrations
- `config/` - Configuration files
- `assets/` - CSS, JS, and other assets
